# Craft Notes

`/craft` output goes here: literature gap analyses, originality assessments, defense preparation notes, and critical feedback on author-written prose.

Each file corresponds to a `/craft` session for a specific section:

```markdown
# Craft Notes: [Section Name]
Date: [date]

## Pre-Writing Briefing
[Literature context, originality check, defense preview]

## Author's Draft
[Reference to draft file/version]

## Critique
[Argument strength, voice analysis, missing engagement, defense readiness]
```

Naming convention: `[section-slug]-[date].md` (e.g., `methodology-2026-03-15.md`)
