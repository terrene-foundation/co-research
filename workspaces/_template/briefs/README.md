# Paper Briefs

Place your paper scope, objectives, and venue targets here. Each brief is a markdown file that describes:

- **Paper title** (working title)
- **Target venue** (arXiv, SSRN, AIES, AI & Society, or other)
- **Research question** (what this paper answers)
- **Contribution** (what is new or different about this work)
- **Scope boundaries** (what this paper does NOT cover)
- **Key arguments** (the main claims, in order)
- **Target audience** (who should read this and what they will learn)
- **Deadline** (if venue has a submission deadline)

Example filename: `my-paper-brief.md`
