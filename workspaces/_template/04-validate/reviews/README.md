# Verification Reports

Verification reports and challenge results go here. Created by `/validate-claim`, `/challenge`, `/check-refs`, and `/preflight` commands.

File types:

- `claim-verification-[target].md` -- Results from `/validate-claim`
- `challenge-[target].md` -- Results from `/challenge`
- `ref-audit-[paper].md` -- Results from `/check-refs`
- `preflight-[paper].md` -- Results from `/preflight`

These reports document the paper's validation history and should be preserved as evidence of rigorous quality control.
