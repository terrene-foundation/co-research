# Literature Notes

Literature notes and paper assessments go here. Created by `/literature` command or manually.

Each file is a structured assessment covering:

- Full citation with DOI or stable URL
- What the paper says (key arguments and findings)
- What the paper does NOT say (common misinterpretations)
- Methodology
- Limitations
- Connection to your paper's argument
- Confidence level in the assessment

Naming convention: `[topic-slug].md` (e.g., `institutional-design-theory.md`)
