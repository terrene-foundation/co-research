# arXiv Submission Rules

## Scope

Documents destined for arXiv submission.

## Quick Rules (Always Active)

1. **No internal references.** No file paths, no TODO markers, no draft comments.
2. **All claims substantiated.** Citation, derivation, or flagged as limitation.
3. **Honest limitations required.** Specific failure modes, not hedges.
4. **LaTeX source required.** arXiv may decline PDF-only.
5. **Category**: Select the primary category appropriate to your paper's field (e.g., cs.CR for security, cs.AI for AI, cs.SE for methodology). Cross-list as appropriate.
